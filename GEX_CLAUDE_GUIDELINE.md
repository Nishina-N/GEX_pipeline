# GEX プロファイル：Claude向け解説ガイドライン

> このドキュメントはGEXプロファイルを解説するClaudeが、計算ロジック・グラフの読み方・分析フレームワークを正確に理解するためのリファレンスです。

---

## 1. GEXとは何か

**GEX（Gamma Exposure）** は、マーケットメーカー（MM）がオプションのデルタ中立ヘッジのために行う原資産売買の圧力を、ストライク別に数値化したものです。

### 計算式（BAWモデル使用）

```
GEX_call(K) = +OI_call × Γ_BAW × 100 × S² × 0.01
GEX_put(K)  = -OI_put  × Γ_BAW × 100 × S² × 0.01
NETGEX(K)   = GEX_call(K) + GEX_put(K)
totalGEX    = Σ NETGEX(K)  ← 全ストライクの合計
```

- **コールは正（+）**: MMはコールを売っているため、価格上昇時に原資産を"売る"ヘッジが必要 → 上値抑制
- **プットは負（-）**: MMはプットを売っているため、価格下落時に原資産を"売る"ヘッジが必要 → 下落加速
- **BAWモデル**: アメリカン・オプションの早期行使を考慮したガンマを使用（BSより正確）

## 2. ⚠️ 最重要：totalGEXとsentimentの誤解

### よくある誤解

> 「totalGEXがマイナスだから、Positive GEX（コール優勢）の状況はない」

**これは間違いです。**

### 正しい理解

`sentiment`（"positive_gamma" / "negative_gamma"）は **totalGEXの符号ではなく、SpotとHVLの位置関係** で決まります：

```python
# 3_extract_levels.py より
sentiment = 'positive_gamma' if spot_price > hvl else 'negative_gamma'
```

**totalGEXがマイナスであっても：**
- 特定ストライクには正のNETGEX（= Call Wall）が存在しうる
- SpotがHVLより上であれば `positive_gamma` 環境となる

### 数値例

```
Strike  NETGEX
18000   +500   ← Call Wall（正）
17500   -200
17000   -800   ← Put Wall（最大負）
16500   -300

totalGEX = 500 - 200 - 800 - 300 = -800（マイナス！）
HVL = 17700付近（符号反転点）
Spot = 17800 → Spot > HVL → sentiment = "positive_gamma" ✅
```

## 3. GEXレベルの定義

| レベル | 定義 | 市場での意味 |
|--------|------|-------------|
| **HVL** (High Volatility Level / Gamma Flip) | NETGEXが正→負に符号反転するストライク（線形補間） | このレベルを跨ぐとMMのヘッジ方向が反転 |
| **Call Wall** | NETGEXが最大正のストライク | 最強の上値抵抗。MMのショートヘッジ圧力が最大 |
| **Put Wall** | NETGEXが最大負のストライク | 最強の下値支持。MMのロングヘッジ圧力が最大 |
| **Transition Zone** | Put Wall〜Call Wallの帯域 | NETGEXの正負が混在する中立ゾーン |

---

## 4. sentimentの解釈

### Positive GEX環境（Spot > HVL）

MMは「ロングガンマ」状態：
- **価格上昇時** → MMがデルタ中立維持のため売り → 上値を抑制
- **価格下落時** → MMがデルタ中立維持のため買い → 下値を支持
- 結果：**価格が安定・レンジ化しやすい**（低ボラ環境）

### Negative GEX環境（Spot < HVL）

MMは「ショートガンマ」状態：
- **価格上昇時** → MMが買い追随 → 上昇が加速
- **価格下落時** → MMが売り追随 → 下落が加速
- 結果：**価格がトレンドを形成しやすい**（高ボラ環境）

## 5. チャート構造の読み方

### 3パネル構成

| パネル | 内容 |
|--------|------|
| **左パネル** | ローソク足 + 出来高 + GEXレベル水平線（HVL・Call Wall・Put Wall） |
| **中央パネル（短期）** | DTE 0〜7日のGEXヒストグラム（横棒） |
| **右パネル（長期）** | 次の2つの月次SQ（第3金曜）のGEXヒストグラム（横棒） |

### 水平線の色

- 🟢 **緑** = Call Wall（上値抵抗）
- 🔴 **赤** = Put Wall（下値支持）
- **点線** = HVL（Gamma Flip）

### ヒストグラムの読み方

- **右向き（正）のバー** = そのストライクでMMは買いヘッジ優勢 → 下値支持
- **左向き（負）のバー** = そのストライクでMMは売りヘッジ優勢 → 上値抵抗

## 6. 🆕 価格方向性・レンジ幅の分析フレームワーク

> GEXプロファイルを読んで「価格がどちらに動きやすいか」「どの程度の幅か」を判断するための手順

### STEP 1: センチメントの確認

**Spot vs HVL の位置関係を確認する。**

| 状況 | 判断 |
|------|------|
| Spot > HVL | **Positive GEX**: 安定・レンジ化バイアス |
| Spot < HVL | **Negative GEX**: トレンド・高ボラバイアス |
| Spot ≈ HVL（直近) | **転換点**: どちらに振れるか要注意 |

---

### STEP 2: Transition Zone内での位置確認

**Spotが Put Wall〜Call Wallのどの位置にいるかを確認する。**

```
Put Wall ←─────────────── Spot ──────────────→ Call Wall
         ↑ put_distance              call_distance ↑
```

| Spotの位置 | 解釈 |
|------------|------|
| Zone上半分（Call Wallに近い） | **上昇余地が限定的**。Call Wall手前で失速しやすい |
| Zone中央付近 | 両方向に均等な引力。方向感が出にくい |
| Zone下半分（Put Wallに近い） | **下落余地が限定的**。Put Wall付近でバウンスしやすい |
| Call Wallを上抜け | 短期強気バイアス。ただし次のWallまでの距離が新たなターゲット |
| Put Wallを下抜け | 短期弱気バイアス。下落加速リスクあり |

---

### STEP 3: レンジ幅の判断（totalGEX × Wall強度）

**totalGEXの絶対値とWallの強度（NETGEX値）を組み合わせて判断する。**

#### totalGEXの大きさとレンジの関係

| totalGEX | 環境 | レンジへの影響 |
|----------|------|----------------|
| 大きな正（例: +10億以上） | Positive GEX 強 | **レンジが非常に狭い**。MMのヘッジ圧力が強く、価格が吸い付けられる |
| 小さな正〜ゼロ近傍 | Positive GEX 弱 | 安定傾向だが、壁は薄い。突発的なブレイクが起きやすい |
| 小さな負〜ゼロ近傍 | Negative GEX 弱 | 方向感が出やすいが、トレンドは長続きしない場合も |
| 大きな負（例: -10億以下） | Negative GEX 強 | **大きなトレンドが発生しやすい**。一方向の値動きが加速する |

#### Wall強度（NETGEX絶対値）と耐久性

| WallのNETGEX絶対値 | 解釈 |
|--------------------|------|
| 大きい（例: 5億以上） | **岩盤ウォール**。容易には抜けない。価格はここで跳ね返されやすい |
| 中程度（1〜5億） | 一定の抵抗感あり。大きなニュースや出来高で抜ける可能性あり |
| 小さい（1億未満） | **薄いウォール**。抜けやすい。抜けた場合の加速に注意 |

---

### STEP 4: 方向性バイアスの総合判断

以下の要素を組み合わせて方向性バイアスを判断する：

#### ✅ Call Wall方向（上昇）に動きやすいケース

- `sentiment = positive_gamma` かつ Spotがzone下半分（Put Wall付近）→ バウンス期待
- `totalGEX` が大きな正 → MMの買いヘッジが強く、下値が固い
- 短期Put WallのNETGEX絶対値が大 → 近くで強いサポート
- Spotがすでに移行してCall Wallを上抜け確定 → モメンタム継続

#### ✅ Put Wall方向（下落）に動きやすいケース

- `sentiment = negative_gamma` → MMのヘッジが下落を増幅
- Spotがzone上半分（Call Wall付近）でCall Wallが強い → 上値抑制で反落リスク
- `totalGEX` が大きな負 → 相場全体がベアガンマ。値動き加速しやすい
- 短期Call WallのNETGEX が小さい → 薄い壁。抜けた後は次のターゲットへ直行

#### ⚖️ 両方向に動きにくいケース（レンジ継続）

- `sentiment = positive_gamma` かつ `totalGEX` が大きな正
- Spotがzone中央付近
- 短期・長期ともにCall Wall / Put Wallが均等に強い

---

### STEP 5: 短期 vs 長期の比較

短期（DTE 0-7日）と長期（月次SQ）のWall位置を比較することで、**足元の圧力と中期的な岩盤**を区別できる。

| 比較の見方 | 解釈 |
|------------|------|
| 短期・長期のCall Wallが同水準 | **強固な上値抵抗帯**。2つの期間でMMヘッジが重なる |
| 短期・長期のPut Wallが同水準 | **強固な下値支持帯**。2つの期間でサポートが重なる |
| 短期Wall ≠ 長期Wall | 短期的なWallは週次SQ後に消滅する可能性。月次SQのWallが中期的な目標水準 |
| 短期に壁がない（0DTE中心） | 日中の動きが大きくなりやすい。週次SQ明けに特に注意 |

## 7. ⛔ ウォール・ブレイク時の大きな価格変動リスク

> **重要**: Wallを突破した場合、一般的な想定以上の値動きが発生するリスクがあります。

### なぜWallブレイク後に加速するのか

1. **MMのヘッジ方向が逆転する**: Call WallをSpotが上抜けると、それまで売りヘッジをしていたMMが、ポジションがITMになったため新たな買いヘッジに転換 → 上昇を加速させる
2. **次のWallまで「壁のない空白地帯」**: 現在の壁を抜けると、次のWallまでNETGEXが小さいゾーンが存在することが多い → 価格が一気にジャンプ
3. **Negative GEX環境への転落**: HVLを下抜けたり、totalGEXが急激にマイナス化するとsentimentが反転 → MMが下落を増幅する側に回る

### ブレイクシナリオの読み方

#### Call Wallを上抜けた場合

```
Call Wall突破 → MMが新たな買いヘッジ積み増し
             → 次のCall Wall 2までの空白地帯を一気に駆け上がる
             → ガンマスクイーズの可能性（短期間で急騰）
```

**注意すべき状況**: Call WallのNETGEXが小さい（薄い壁）＋大量出来高やカタリスト（決算・マクロイベント）

#### Put Wallを下抜けた場合

```
Put Wall突破 → MMが新たな売りヘッジ積み増し
            → 次のPut Wall 2までの空白地帯を一気に下落
            → 場合によってはHVL割れ → Negative GEX環境に転落
            → 下落が自己強化的に加速するリスク
```

**特に危険な状況**: `sentiment = negative_gamma` の状態でPut Wallまで接近している場合

### ブレイクの「本物か？ダマシか？」を判断するポイント

| 指標 | 本物のブレイク | ダマシ（フェイクアウト） |
|------|--------------|------------------------|
| 出来高 | 急増（通常の2倍以上） | 低出来高 |
| 価格の維持 | 終値でWall水準を上/下回る | 引けにかけて戻る |
| 翌日の継続 | 翌朝もWallの外側でスタート | 翌朝にWall内に戻る |
| GEX変化 | 翌日のGEX更新でWall位置が移動 | Wallの位置が変わらない |

## 8. 解説例文テンプレート

### Positive GEX環境（レンジ想定）

```
現在の環境: Positive GEX（SpotがHVL [価格] を上回っています）

現在のSpot [価格] は、Call Wall [価格] とPut Wall [価格] の間に位置しており、
Transition Zone [幅] の [上/中/下] 半分にあります。

totalGEX [数値] のように大きな正値を示しており、MMのヘッジ圧力が強く働いています。

短期的には、[Call Wall 価格] が上値の壁、[Put Wall 価格] が下値の支持として機能します。
ただし、これらのWallを明確に抜けた場合には値動きが加速するリスクがあります。

[短期 / 長期] Call Wall [価格] のNETGEXは [数値] と[大きく/小さく]、

```


## 9. よくある誤解まとめ

| 誤解 | 正しい解釈 |
|------|-----------|
| totalGEXがマイナス → Positive GEXにはなれない | **誤り**。sentimentはSpot vs HVLで決まる |
| Call WallはSpotの上にある | **誤り**。NETGEXが最大正のストライク。SpotがCall Wallを超えている場合もある |
| Put WallはSpotの下にある | **誤り**。NETGEXが最大負のストライク。構造によってはSpotより上の場合もある |
| totalGEXが大きいほど上昇しやすい | **誤り**。totalGEXが大きな正 = 安定・レンジ化。大きな負 = トレンド化 |
| Wallを超えたら即座に反転 | **誤り**。薄い壁なら突破してそのまま次のWallまで走ることがある |
| 短期と長期のWallは常に一致する | **誤り**。時間軸で異なる場合が多い。両方確認が必要 |
| HVLは固定されている | **誤り**。日次でオプションのOI・IVが変化するため、毎日更新される |
